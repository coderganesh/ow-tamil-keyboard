# ow-tamil-keyboard
A simple, easy to use Tamil keyboard layout for Humans.

## Installation
1. Download **OwLayout.zip** from the latest release
2. Extract it and run the **setup.exe**

## Features
1. Supports both 64 bit and 32 bit
2. Windows OS compatible
3. Easy to install and use
4. No need to memorize key mappings
5. Natural mapping with English letters
6. Only uses 28 keys (26 Alphabetic keys, 1 Shift & 1 AltGr)

## Usage
Each Tamil letter is mapped to the English letter having similar pronounciation.
### உயிரஂ எழுதஂதுகளஂ (Vowels):
அ -> A \
இ -> E \
உ -> U \
எ -> F \
ஐ -> I \
ஒ -> O \
ஔ -> Q \
Shift + அ = ஆ \
Shift + இ = ஈ \
Shift + உ = ஊ \
Shift + எ = ஏ \
Shift + ஒ = ஓ

### மெயஂ எழுதஂதுகளஂ (Consonants):
க -> K \
ங -> G \
ச -> C \
ஞ -> J \
ட -> T \
ண -> B \
த -> D \
ந -> H \
ப -> P \
ம -> M \
ய -> Y \
ர -> R \
ல -> L \
வ -> V \
ழ -> Z \
ள -> W \
ற -> X \
ன -> N \
ஂ -> S

### உயிரஂ மெயஂ எழுதஂதுகளஂ (Syllables):
AltGr = Right Alt (or) Ctrl + Alt

AltGr + உயிரஂ எழுதஂது = உயிரஂ மெயஂ எழுதஂது

### ஆயஂத எழுதஂது (Triple dot consonant):
Shift + ஂ = ஃ

## Examples
1. D + M + (AltGr + E) + Z + S = தமிழஂ
2. E + M + Y + N + S = இமயனஂ
3. P + (Shift + S) + X + (AltGr + U) + W + (AltGr + E) = பஃறுளி

## Customization
You can add new non-Tamil letters to the keyboard layout by importing **OwLayout.klc** in the Microsoft Keyboard Layout Creator software.
